# 24 Solar Terms Widget (Sekki Widget)

A simple, minimalist widget that shows the current **Japanese 24 Solar Terms (Nijūshi Sekki)** in English.
Embed it in Notion or your blog to stay in tune with the seasonal rhythm.

---

## Tech Stack

- HTML / CSS / JavaScript
- GitHub Pages (hosting)

---

## What it shows

- Automatically displays the current solar term based on today’s date
- Includes a short seasonal note and a gentle self-care suggestion
- Switches terms using an approximate 15-day interval logic
  - (Future improvement idea: higher accuracy via an astronomy library or API)
- Calm, neutral design that blends well into blogs and dashboards
- Minimal layout that looks good on mobile

## How to embed

### Embed in Notion

1. Open the Notion page where you want the widget
2. Type `/` and search for **Embed**
3. Select **Embed**
4. Paste the URL below and choose “Embed link”

URL: `https://hitomisty.github.io/sekki-widget-en/`

Tips:
- The widget updates automatically every day
- Great for dashboards, daily notes, or a home page

---

### Embed in a blog / website

Copy and paste this iframe into your HTML:

```html
<iframe
  src="https://hitomisty.github.io/sekki-widget-en/"
  width="350"
  height="300"
  frameborder="0"
  style="border-radius: 12px;"
></iframe>
```

Adjust the size:
- Change the width and height values as needed
